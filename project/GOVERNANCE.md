# Governance

## Editorial Model

Anyone may contribute.

Only the repository owner may approve changes for publication.

## Branches

- main – published content
- develop – integration branch
- feature/* – individual work

## Merge Requirements

- Pull request
- Passing CI
- Code owner approval
- Resolved conversations

## Source of Truth

Markdown in this repository is authoritative.
Generated PDFs, DOCX, and websites are derived artifacts.
