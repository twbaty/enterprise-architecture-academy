# Project Philosophy

Enterprise Architecture Academy exists to teach people how to think like Enterprise Architects.

Certification is a milestone, not the destination.

The repository is treated as an engineering project:
- Markdown is the canonical source.
- Every important decision is documented.
- Documentation evolves through pull requests.
- Published artifacts are generated from Markdown.
- The owner retains final editorial authority.

Success is measured by whether graduates can solve enterprise problems, not whether they can quote a framework.
