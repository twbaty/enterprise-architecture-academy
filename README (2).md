# Enterprise Architecture Academy

## Mission
Build the highest-quality, community-driven curriculum for Enterprise Architecture while maintaining a single editorial vision.

## Goals
- Teach Enterprise Architecture, not just TOGAF.
- Prepare students for Foundation and Practitioner certifications.
- Produce reusable instructor and student material.
- Keep all source material in Markdown.
- Generate PDFs, DOCX, and the website from the same source.

## Guiding Principles
1. Teach the problem before the framework.
2. Architecture supports decisions.
3. Examples are persistent and realistic.
4. Learning beats memorization.
5. Every accepted contribution improves the curriculum.

## Repository Structure

- docs/project – governance and project documentation
- docs/foundation – Foundation curriculum
- docs/practitioner – Practitioner curriculum
- docs/case-studies – recurring scenarios
- docs/templates – reusable authoring templates
