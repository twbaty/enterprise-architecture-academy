# Enterprise Architecture Academy

This is a living Enterprise Architecture curriculum and study resource.

The project has two goals:

1. Help serious learners prepare for the TOGAF Enterprise Architecture Foundation and Practitioner certification path.
2. Teach the architectural judgment required to do the work, not merely recall framework terminology.

## What this repository is

- A transparent record of the material used to study and teach Enterprise Architecture
- A curriculum design project
- A collection of lessons, exercises, cases, and assessments
- A community-contributed project under owner editorial control

## What it is not

- Official Open Group training
- An exam dump
- A collection of recalled certification questions
- A substitute for the official TOGAF Standard
