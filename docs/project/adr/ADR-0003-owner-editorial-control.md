# ADR-0003: Open contribution with owner editorial control

**Status:** Accepted

## Context

Community input can improve accuracy and breadth, but uncontrolled merging would fragment the curriculum's scope, sequence, and voice.

## Decision

Anyone may propose changes. The repository owner must approve accepted changes and retains final authority over merge and publication decisions.

## Consequences

- Contributions remain open.
- Editorial coherence is preserved.
- Code Owner and branch rules must enforce owner approval.
- Acceptance and release timing remain separate decisions.
