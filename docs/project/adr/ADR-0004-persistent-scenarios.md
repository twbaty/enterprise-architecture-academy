# ADR-0004: Persistent scenarios support cumulative learning

**Status:** Accepted

## Context

Disconnected examples teach isolated concepts but do not show how architecture decisions accumulate, interact, and create downstream consequences.

## Decision

The curriculum uses persistent fictional enterprises, led by Northstar Regional Health, across multiple lessons and courses.

## Consequences

- Learners retain context between lessons.
- Earlier decisions can create later constraints and trade-offs.
- Scenario facts require version control and internal consistency.
- Sector variants are still used when comparison adds value.
