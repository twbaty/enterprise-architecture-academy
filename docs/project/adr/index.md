# Architecture Decision Records

Architecture Decision Records preserve the reasoning behind important project decisions.

## Status values

- **Proposed** — under consideration.
- **Accepted** — currently governs the project.
- **Superseded** — replaced by a later ADR.
- **Deprecated** — retained for history but no longer recommended.

## Required sections

Each ADR contains:

- status;
- context;
- decision;
- consequences;
- superseding relationship, when applicable.
